# Protein Structural Comparison and Classification using a Linear Algebra Approach
The function of a protein depends on its structure and folding pattern. One of the main forces helping it folds is the hydrophobic interaction. Here we aim to compare and classify protein structures into fold families based on pattern of hydrophobic intrachain interactions using Singular Value Decomposition. 

## Open Science
We reproduced this project based on work published by [Gomide et al. (2009)](https://pmc.ncbi.nlm.nih.gov/articles/PMC3036040/).
All proteins used in this project are retrieved from the [RCSB Protein Data Bank](https://www.rcsb.org/).
